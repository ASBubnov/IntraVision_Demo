Проект ASP.NET MVC,бд используется PgSql, для подключения к базе используется ORM EF, для управления зависимостями используется DI от майкрософта, на фронте используется jQuery на пользовательском интерфейсе, а на админском просто чтоб показать свои навыки использовал тэг form.
Секретный ключ и Id машины хранятся в конфигурационном файле. Решение является универсальным и масштабируемым, при необходимости может использоваться на нескольких автоматах, при этом данные будут храниться в одной СУБД без конфликтов, что обеспечивается полем в конфигурационном(VendingMachineId).
Автомат выдерживает перезагрузку, в случае перезапусков автомат не забывает баланс пользователя.
Из того что можно улучшить: Редактирование на админской панели происходит сразу, лучше сделать поля readonly и добавить кнопки для открытия отдельных страниц редактирования.
